# 📚 practicas-js

Repositorio de ejercicios y prácticas de **JavaScript**. Aquí documento mi progreso aprendiendo el lenguaje, desde conceptos básicos hasta temas más avanzados.

## 🎯 Objetivo

Mejorar mis habilidades en JavaScript a través de la práctica continua, resolviendo ejercicios y construyendo pequeños proyectos.

## 📁 Estructura del repositorio

```
practicas-js/
├── 01-basicos/         # Variables, tipos, operadores
├── 02-condicionales/   # if/else, switch, ternarios
├── 03-bucles/          # for, while, forEach
├── 04-funciones/       # Funciones, arrow functions, callbacks
├── 05-arrays/          # Métodos de array (map, filter, reduce)
├── 06-objetos/         # Objetos, destructuring
├── 07-dom/             # Manipulación del DOM
├── 08-async/           # Promesas, async/await, fetch
└── 09-proyectos/       # Mini proyectos aplicando lo aprendido
```

## 📝 Temas cubiertos

- [x] Sintaxis básica

## 🚀 Cómo usar este repo

```bash
# Clonar el repositorio
git clone https://github.com/ninjadiego/practicas-js.git

# Entrar al directorio
cd practicas-js

# Ejecutar un archivo con Node
node 01-basicos/variables.js
```
